Media Queries - Abordagem Mobile First

Conceito

São regras especiais em que criamos alterações na página web de acordo com determinadas condições.

Também conhecidas como breakpoints (pontos de adaptação).

Aplicação

Media Queries são aplicadas de acordo com a estratégia usada para design responsivo:

  • Mobile First: telas pequenas para grandes
  • Desktop First: telas grandes para pequenas
